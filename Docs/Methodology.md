# Methodology

This portfolio follows a standard data analysis workflow:

1. Data Collection
Datasets were obtained from public data repositories such as Kaggle.

2. Data Cleaning
Missing values were checked and corrected. Data types were standardized.

3. Exploratory Data Analysis
Statistical summaries and visualizations were used to explore patterns.

4. Visualization
Charts were created using Matplotlib and Seaborn to communicate insights.

5. Insight Generation
Each project concludes with key findings and recommendations.

Tools used:

Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook.
