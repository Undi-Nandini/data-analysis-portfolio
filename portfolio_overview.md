# Data Analysis Portfolio Overview

This portfolio contains five real-world data analysis projects across different domains. 
The goal of this portfolio is to demonstrate data cleaning, exploratory data analysis, 
statistical analysis, visualization, and business insight generation using Python.

Projects included:

1. Supermarket Sales Analysis (Retail Domain)
2. Student Performance Analysis (Education Domain)
3. Weather Data Analysis (Climate Domain)
4. Heart Disease Risk Analysis (Healthcare Domain)
5. Apple Stock Market Analysis (Finance Domain)

Each project includes:

- Data cleaning and preprocessing
- Exploratory data analysis
- Statistical analysis
- Data visualizations
- Business insights and recommendations
- Professional report with executive summary

Tools used: Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook.
